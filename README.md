## ğŸ“ 1. Penjelasan Nano
**Nano** adalah editor teks berbasis terminal yang mudah digunakan di Linux.  
Untuk membuka file dengan Nano, gunakan perintah:
```bash
nano nama_file
```
Tampilan antarmuka nano

